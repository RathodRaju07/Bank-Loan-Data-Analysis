# Bank-Loan-Data-Analysis
This project focuses on analyzing bank loan data to understand customer behavior, loan performance, and risk patterns. Using SQL for data extraction and transformation and Power BI for visualization, the project provides actionable insights to support better lending and risk management decisions.
